# COVID-19 Global Data Analysis

## Overview
Analysis of global COVID-19 data across 200+ countries covering
cases, deaths, vaccinations, and trends from 2020 onwards.

## Tools Used
- Python (pandas) — data cleaning and feature engineering
- MySQL — advanced SQL with window functions and CTEs
- Tableau — 5-visual interactive dashboard with world map

## Advanced Skills Demonstrated
- SQL Window Functions (7-day rolling average, PARTITION BY)
- SQL CTEs (WITH statements)
- Geographic map visualisation in Tableau
- Dual-axis trend charts
- Scatter plot correlation analysis

## Key Findings
- USA led globally with 103.4M cases; Brazil had the highest death rate 
  among top 10 countries at 1.85%
- China reported 99.4M cases but an exceptionally low death rate of 0.12%
  — the lowest among top 10 nations
- 2022 was the worst year globally with 444M new cases — nearly double 2021
- Cases dropped 90% from 2022 to 2023 as immunity and vaccinations took effect
- Gibraltar achieved 110% vaccination rate (boosters counted separately)
  while Burundi had only 0.27% — the starkest global inequality in the dataset
- Africa had the most countries affected (58) but the lowest recorded cases
  (4M) — likely reflecting undertesting rather than true low spread
- South America had the second highest death toll despite only 14 countries

Our World in Data COVID-19 Compact Dataset
